# 🔧 Plomberie Pro — Application

This is the main application directory for **Plomberie Pro**.

- **Backend**: Laravel 12
- **Frontend**: React 18 + Inertia.js
- **Styling**: Tailwind CSS 4

## 📖 Documentation

For full project documentation, including installation steps and feature overview, please refer to the **[Root README](../README.md)**.

## 🚀 Quick Commands

```bash
# Install dependencies
composer install
npm install

# Start development
npm run dev
php artisan serve
```
